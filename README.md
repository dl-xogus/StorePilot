# 🚀 StorePilot

AI 기반 매장 관리 및 데이터 분석 플랫폼  
매장의 매출, 메뉴, 재고, 스케줄 데이터를 분석하여 효율적인 운영을 지원합니다.

---

## 📌 프로젝트 소개

StorePilot는 매장 운영 데이터를 기반으로  
AI 분석 및 시각화를 제공하는 스마트 매장 관리 서비스입니다.

### 주요 기능
- 📊 매출 데이터 분석
- 🍽 메뉴 판매량 관리
- 📦 재고 관리
- 👨‍💼 직원 스케줄 관리
- 🤖 AI 기반 운영 인사이트 제공
- 📈 데이터 시각화 대시보드
